# 📝 게임 개발 사양서 (Full Specification) - BGM 타임라인 & 비디오 통합본

## 1. 프로젝트 개요 및 프레임워크 세팅

- **게임 장르:** 2D 벨트스크롤 러너 + 어그로 군단 관리 액션 게임 (BGM 타임라인 동기화)
- **개발 프레임워크:** **Phaser 3** (최신 ES6+ 문법 스타일 적용)
- **물리 엔진:** Arcade Physics (기본 중력 `y: 0`, 충돌 및 속도 제어 중심)
- **배포 환경 및 파일 구조:**
    - AI 개발 생산성 극대화 및 환경 설정 오류 방지를 위해 **단일 `index.html` 파일**로 개발을 진행한다.
    - Vercel 및 GitHub Pages 정적 배포 목적 (서버 없는 HTML5 단일 파일 구조 가능)
- **해상도 및 캐릭터 규격:**
    - 해상도: 800x400, `pixelArt: true` (도트 흐려짐 방지), `Scale.FIT` 및 `CENTER_BOTH` 적용
    - 캐릭터 크기: **32x32 픽셀 규격**으로 고정한다.
- **리소스 및 비주얼 형태:**
    - 모든 비주얼(캐릭터, 아이템, 배경, 절벽 등)은 임시로 **단색 사각형 도형**으로 구현하며, 고유 색상 스키마를 설정한다. 추후 스프라이트 및 텍스처로 대체하기 쉽도록 구현한다.
    - 비디오 에셋(`opening.mp4`, `ending_normal.mp4`, `ending_true.mp4`) 및 BGM 음원은 외부에서 로드하되, 로딩 실패 시 자동으로 **도형 기반 시네마틱 연출(폴백) 및 Web Audio API 비트음**으로 연동된다.

### 📁 씬(Scene) 구조 및 아키텍처
단일 파일 내에서 역할을 다음과 같이 분리된 클래스로 관리한다.
```javascript
class BootScene extends Phaser.Scene { /* 에셋(도형 텍스처 등록) 로드 및 초기 세팅 */ }
class TitleScene extends Phaser.Scene { /* 시작 타이틀 화면 (BGM 로딩 지원) */ }
class IntroScene extends Phaser.Scene { /* 오프닝 영상 연출 (자동재생 정책 대응 및 실패 시 자막 폴백) */ }
class PlayScene extends Phaser.Scene { /* 게임 본편 (STAGE 1 ~ 3 타임라인 진행) */ }
class EndingScene extends Phaser.Scene { /* 에필로그 연출 및 최종 엔딩 시네마틱 (비디오 또는 폴백 도형 물리 연출) */ }
```

## 2-1. 게임 시스템 및 진행 방향 (좌향 이동)

이미지 분석(`EX1.png`, `EX2.png`) 결과에 따라 본 게임의 진행 구조는 다음과 같이 설정된다.
- **주인공 진행 방향:** 주인공은 **왼쪽(좌측)을 바라보며 달리며**, 배경 타일은 **오른쪽(우측)으로 롤링 스크롤**된다.
- **좀비 군단 위치:** 좀비들은 플레이어의 뒤인 **화면 우측 1/3 구역**에 군단(`GROUPING` 상태)을 형성하여 쫓아온다.
- **절벽 지형 위치:** 게임 시간 270초(4분 30초)가 되어 에필로그가 시작되면 스크롤이 즉시 멈추고 **화면 최좌측(X: 0 지점)에 낭떠러지 절벽**이 고정 생성된다.

## 2-2. 스테이지 타임라인
BGM의 재생 시간(`audio.currentTime` 또는 타이머)에 맞춰 스테이지가 자동으로 진행된다. 스테이지 전환 시 1초간 페이드 아웃 후 새로운 스테이지 설정으로 페이드 인된다.

1. **Stage 1: 저녁 길가 (0초 ~ 90초)**
   - 배경 색상: 노을빛(0xd88d55) / 스크롤 속도: 기본(1.5) / 좀비 스폰 주기: 2.0초
2. **Stage 2: 폐허 (90초 ~ 180초)**
   - 배경 색상: 주황/남색(0x3a4f7c) / 스크롤 속도: 빠름(2.0) / 좀비 스폰 주기: 1.5초
3. **Stage 3: 공사장 (180초 ~ 270초)**
   - 배경 색상: 짙은 남색 밤하늘(0x111625) / 스크롤 속도: 매우 빠름(2.5) / 좀비 스폰 주기: 1.0초
   - **좀비 플레이 및 무적 상태**: 180초 도달 시 플레이어가 완전히 좀비 상태(`player_infected`)로 변이되어 질주합니다. 이 스테이지에서는 인간성이 0이 되어도 게임 오버되지 않으며, 절벽에 도달할 때까지 좀비 공격으로부터 완전히 무적 상태(피해 및 넉백 없음)로 기동합니다.
   - 건설 크레인, 펜스, 드럼통, 나무상자 등 장애물이 등장하며 분홍색 'C' 아이템(백신/약)이 상공에 출현합니다.
4. **Stage 4: 공사장 절벽 (270초 ~ 291초 / 에필로그)**
   - 4분 30초(270초) 도달 시 플레이어의 조작권이 강제 박탈되며 화면 스크롤이 정지하고 최좌측에 절벽이 생성된다.
   - 플레이어는 21초 동안 절벽 방향(좌향)으로 터덜터덜 걷는 애니메이션으로 자동 이동하고, 좀비 떼가 바로 뒤에서 밀착해 따라온다.
   - 정확히 4분 51초(291초)가 되었을 때 엔딩 분기로 진입한다.

## 👤 3. 플레이어(Player) 세부 사양

### 1. 기본 데이터 및 입력
- `maxHumanity`: 100 / `currentHumanity`: 100 (초당 자연 감소, 알약 획득 시 20 회복)
- **PC 조작:** 상하좌우 방향키로 이동 (이동 속도: 120), 스페이스바로 전방 야구배트 공격(히트박스 생성)
- **모바일 조작:** 화면 왼쪽 절반 터치 시 터치한 위치로 상하좌우 이동 / 오른쪽 절반 터치 시 배트 타격
- **공격 쿨타임:** 0.4초, 전방 타격 범위 `width: 60px, height: 40px`

### 2. 인간성 수치에 따른 외형 변화 (FSM) 및 초상화 UI
- **인간성 상태별 왼쪽 하단 초상화(Portrait) UI 변화**:
  - **100% ~ 51% [HUMAN]**: 초상화가 하늘색(0x00d2fc)으로 채워지며 정상 인간 상태 표시.
  - **50% ~ 26% [INFECTING]**: 초상화가 옅은 민트색(0x5cc8a6)으로 변하며 감염 진행 중 표시.
  - **25% ~ 1% [INFECTED]**: 초상화가 탁한 녹색(0x2c7a5f)으로 변하며 감염 완료 상태 표시.
  - **0% [ZOMBIE]**: 초상화가 완전한 좀비 녹색(0x2ebd59)으로 변하며 좀비화 완료 상태 표시.
- **인게임 캐릭터 액션 스프라이트 (2가지 상태만 존재)**:
  - **인간 모드 (Human Mode)**: 플레이 도중 인간성 수치가 1 이상이고 Stage 1 & 2일 때는 항상 하늘색(`player_normal`) 상태 유지.
  - **좀비 모드 (Zombie Mode)**: 마지막 Stage 3(180초 이후)에 도달하거나, Stage 1 & 2 도중 인간성이 0이 되는 패배 순간, 또는 270초 에필로그 절벽 시퀀스 진행 중에는 녹색 좀비형(`player_infected`) 상태로 변경.
- **0% [DEAD_END] (Stage 1 & 2)**: 일반 플레이(0~180초) 도중 인간성이 0이 되면 즉시 사망 처리. 제어권이 박탈되고 플레이어 사각형이 쓰러지며(-90도 회전) 주위 좀비 떼가 대거 스폰되어 플레이어를 덮치는 컷씬 연출 후, [배드 엔딩: 좀비화 감염] 화면이 뜬다. (절벽으로 이동하지 않음)
- **0% (Stage 3)**: 마지막 스테이지(180~270초)에서는 인간성이 0이 되어도 게임오버가 작동하지 않고 절벽 도달 시점까지 무적으로 계속 플레이할 수 있습니다.

### 3. 피격 및 야구공 투척
- **피격 무적 프레임:** 좀비와 충돌 시 인간성이 10 감소하며, 캐릭터가 1초간 깜빡이며 뒤로 약간 넉백되고 무적 판정이 된다.
- **야구공(원거리 무기):** 필드에서 야구공 아이템 획득 시 3회에 한해 공격이 원거리 투척 공격(사각형 투사체 발사)으로 전환된다. 투사체는 좀비 타격 시 좀비를 무력화하여 무리로 합류시킨다.

## 🧟 4. 좀비(Zombie), 보급 상자 및 NPC 세부 사양

물리 연산 렉 방지를 위해 모든 좀비는 서로 충돌(Collide)하지 않고 중첩(Overlap)이 허용된다.

### 1. 좀비 상태 (State) 머신
- **`CHASING` (추적)**: 플레이어를 타겟으로 상하좌우 실시간 추격.
- **`KNOCKBACK` (넉백)**: 피격 시 이동 방향 반대(왼쪽)로 0.2초간 강제 밀려났다가 다시 추격을 시작함.
- **`DEAD_FLY` (사망 연출)**: 체력 소진 시 물리 충돌 제거 후 공중 회전하며 포물선을 그리며 화면 오른쪽 밖으로 날아감.
- **`GROUPING` (무리 합류)**: 사망하여 화면 오른쪽 끝으로 날아간 좀비는 소멸하지 않고, 화면 우측 `X: 850` 구역에서 플레이어를 뒤따르는 군단 무리에 자연스럽게 합류함 (기존 엘리트 등의 크기/색상 틴트 상태 유지).

### 2. 좀비 종류 (`ZOMBIE_TYPES`) 및 특수 연출
- **NORMAL (일반 좀비)**: 녹색(0x2ebd59) 사각형. 속도 60.
  - **엘리트 좀비 (확률적 스폰)**: 일반 좀비 스폰 시 15% 확률로 엘리트 좀비로 등장. 붉은색 틴트(0xff5555) 및 1.25배 확대된 스케일을 가짐. 속도는 78로 설정되며 체력(HP)은 2~3으로 야구배트로 타격 시 넉백되어 다시 쫓아옴.
- **RUNNER (돌진 좀비)**: 적색(0xff3f3f) 사각형. 화면 왼쪽/오른쪽에서 랜덤 등장 가능. 스폰 전 화면 경계에 빨간색 느낌표(`!`) 경고창이 3번 깜빡인 후 플레이어를 향해 360의 빠른 속도로 돌격함.
- **STALKER (상자 경비 좀비)**: 어두운 회색(0x4f4f4f) 및 형광 하늘색 눈. 보급 상자(Box) 스폰 시 세트로 생성되는 첫 번째 좀비. 상자에 밀착하여 함께 우측으로 이동함 (속도 48).
- **SPITTER (원거리 침 뱉는 좀비)**: 보라색(0x8e44ad) 및 주황색 입. 화면 왼쪽에서 스폰되어 보급 상자와 동일하게 도로 스크롤 흐름 속도(vx = 60)로 흘러가며, 화면 내에 있는 동안 2초마다 플레이어를 향해 연두색 독침 투사체(`spit_acid`)를 발사함. 독침 피격 시 플레이어의 인간성이 10 감소하고 무리가 3마리 탈탈 털려나가는 피격 패널티가 동반되며, 날아오는 독침은 야구배트로 휘둘러서 파괴할 수 있음.

### 3. 유기적인 좀비 군단(Horde) 포메이션
- 무리에 합류한 좀비들은 일렬로 경직되게 늘어서지 않고, 유기적이고 자연스러운 무리 느낌을 주도록 설계함.
- 각 좀비마다 **개별 랜덤 오프셋(X, Y)** 및 **사인파 기반의 유동성 좌우/상하 흔들림**, **속도의 미세한 가감(주기적 흐름)** 연출을 프레임별로 적용함.

### 4. 보급 상자(Box) 세트 스폰 및 파괴 전리품
- **보급 상자 스폰**: 10초마다 화면 왼쪽에서 스폰되어 스테이지 스크롤 속도로 이동함. 스폰 시 상자 주위에 2~3마리의 좀비가 한 세트로 함께 배치되어 상자에 고정 오프셋으로 붙어 이동함.
- **상자 파괴**: 야구배트로 2회 타격 시 파괴(HP 2)되며, 파괴 시 아래의 확률로 전리품이 결정됨:
  - **좀비 급습 (20%)**: 상자 속에서 좀비가 튀어나와 즉시 플레이어를 추격함.
  - **알약 출현 (30%)**: 획득 시 인간성 게이지를 20 회복시켜 주는 알약 아이템 생성.
  - **시민 구조 (50%)**: 구출 시 하트/별 연출과 함께 우측 화면 밖으로 도망치는 시민 스폰:
    - **일반 시민 (98%)**: 하늘색 셔츠의 일반 시민 (♥ 아이콘).
    - **심통봇 (1%)**: 보라색 몸체와 노란 안테나를 지닌 특수 캐릭터 (★ 아이콘).
    - **스네이크 (1%)**: 국방색 밀리터리 복장과 안대를 착용한 특수 캐릭터 (★ 아이콘).

### 5. 연구원(Scientist) 고정 주기 리젠 및 예외 처리
- **연구원 상자**: 1분 ± 10초(50~70초) 주기마다 고정적으로 하나의 보급 상자가 "연구원 탑재 상자"(`containsResearcher = true`)로 지정되어 스폰됨.
- **구조 방법**: 해당 상자를 깨뜨리면 흰색 의사가운을 입은 연구원(Scientist)이 구조됨.
- **예외 처리 (놓침)**: 플레이어가 해당 상자를 파괴하지 못하고 화면 오른쪽 끝으로 지나쳐 보내 소멸하는 경우, 추가적인 재기회 없이 연구원 구조 기회는 소멸합니다 (놓치면 그대로 실패).

## 🎬 5. 최종 엔딩 분기 구조 (291초 진입)

정확히 291초에 절벽 끝에 도달했을 때, 축적된 데이터를 비교하여 엔딩이 결정된다.

```javascript
function checkEndingCondition(gameData) {
    if (gameData.survivedTime >= GAME_CONFIG.epilogueTime &&
        gameData.pillsCollected >= GAME_CONFIG.minPillsForTrueEnding &&
        gameData.rescuedScientists >= 3 && // 연구원 3명 이상 구출 필수
        gameData.maxZombieCombo >= GAME_CONFIG.minComboForTrueEnding) {
        return "TRUE_ENDING";
    }
    return "NORMAL_ENDING";
}
```

### 엔딩 연출 및 조건
- **진엔딩 (True Ending) 조건**: 알약 획득수 만족, 좀비 최대 콤보 만족, 그리고 **연구원 구조 수가 3명 이상**(`window.gameData.rescuedScientists >= 3`)이어야 함.
- **일반 엔딩 (Normal Ending) 조건**: 위 조건 중 하나라도 미달될 시, 절벽 아래로 좀비 떼와 함께 낙하하는 일반 엔딩으로 분기됨.
- **시네마틱 및 폴백 연출**: 비디오 로딩 여부에 따라 고유 비디오가 재생되거나 도형 물리 연출(헬기 낚아채기 vs 절벽 낙하)이 구동됨.
